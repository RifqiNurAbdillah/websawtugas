Tentang

Web ini dibuat sebagai bagian dari tugas mata kuliah Decision Support System (DSS). Tujuan dari pembuatan web ini adalah untuk membantu proses pencatatan dan pengelolaan data secara efektif dan efisien, sehingga memudahkan pengambilan keputusan berdasarkan informasi yang tersimpan. Dengan menggunakan web note ini, diharapkan pengguna dapat dengan mudah mencatat, menyimpan, dan mengakses data penting yang dibutuhkan dalam pengambilan keputusan yang tepat dan cepat.
